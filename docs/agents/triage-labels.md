# Triage Labels

工程 skills 使用五个标准角色。右侧字符串是在 GitHub Issues 中实际使用的标签。

| Skill role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可由独立 Agent 领取 |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 使用角色名称时，应用本表对应的 GitHub 标签。
